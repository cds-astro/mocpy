Install¶

MOCPy relies on the following mandatory external dependencies :

  • cdshealpix

  • astropy

  • matplotlib

  • networkx used in get_boundaries

  • lark-parser used for serializing the sky coverages to ASCII format

It has also a few optional dependencies that are mainly used in deprecated methods and will be removed in the future:

  • astropy-healpix is used by the old plotting methods as well as in get_boundaries

To upgrade the mocpy package to the latest version:

pip install mocpy -U

To install mocpy type:

pip install mocpy
